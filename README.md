# Trainingslog

A simple and secure training log app for tracking workouts and progress.

## Features
- Track workouts and exercises
- Simple and clean interface
- Works as a PWA (installable on mobile)
- Offline support via service worker

## Tech
- HTML
- JavaScript
- Service Worker (PWA)
